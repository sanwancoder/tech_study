# 快捷键

##windows环境下

| 快捷键 | 快捷键作用|
|--------| ------ |
|Ctrl+G|跳转到指定行|
|Ctrl+P| 快速打开最近打开的文件|
|Ctrl+Shift+F|选中目录全局搜索|
|F11|全屏|
|Alt + ←|回到上次编辑的地方|
|(Ctrl + K , Ctrl + S)先摁下Ctrl + K，再摁 Ctrl + S|调出全部快捷键|
